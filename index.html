Diffrence between local Storage and session Storage

Session storage is only available for the duration of the browser session. Page sessions are valid for only one tab at a time. Then a document is loaded in a browser tab, a unique page session is created and assigned to that tab. That page session is only valid for that specific tab. A new session is created each time a tab or window is opened.
Local storage is the opposite of session storage. Local storage store date in your browser after using the browser. Data will not delete a browser tab or a unique page session. Local storage data will not delete until we delete data manually.
